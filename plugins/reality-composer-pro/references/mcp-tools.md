# RealityComposerPro local MCP helper tools

The bundled helper exposes these local tools:

- `rcp_doc_search`
- `rcp_reference_get`
- `rcp_docs_sidebar_map`
- `rcp_release_notes_plan`
- `rcp_mcp_candidates`
- `rcp_project_doctor`
- `rcp_asset_audit`
- `rcp_swift_scaffold`
- `rcp_generate_checklist`
- `rcp_plan_task`
- `rcp_gitignore_suggestions`

Use the helper for project inspection, planning, scaffolding, and handoff. It is not an Apple remote-control API for the Reality Composer Pro app.
