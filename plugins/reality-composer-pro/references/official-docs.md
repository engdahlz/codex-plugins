# Official sources

Use these sources before third-party snippets:

- Apple Reality Composer Pro product page: `https://developer.apple.com/reality-composer-pro/`
- Apple RealityKit documentation: `https://developer.apple.com/documentation/realitykit/`
- Apple visionOS documentation: `https://developer.apple.com/documentation/visionos/`
- Apple WWDC videos: `https://developer.apple.com/videos/`
- Apple `reality-composer-pro-plugin` Swift package: `https://github.com/apple/reality-composer-pro-plugin`
- Apple AR tools and USDZ guidance: `https://developer.apple.com/augmented-reality/tools/`

Reality Composer Pro 3 and the editor-plugin package can be SDK/beta sensitive. Always check the installed Xcode version and package availability before final code.
