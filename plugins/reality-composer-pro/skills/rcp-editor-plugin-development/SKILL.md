---
name: rcp-editor-plugin-development
description: Build custom Reality Composer Pro 3 editor plugins using Apple's Swift package when available.
---

Use Apple's `reality-composer-pro-plugin` package as the source of truth. Keep serializable shared component types `Codable` and visible to both the app and editor plugin target. Register custom components, systems, and actions in the editor plugin. Verify the installed Xcode/RCP SDK before final code.
