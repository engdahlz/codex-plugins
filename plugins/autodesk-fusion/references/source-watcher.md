# Source watcher

Fusion, APS, MCP, and Codex documentation changes quickly. Use `scripts/check_autodesk_sources.py` to detect source status changes.

Tracked areas:

- Fusion API overview.
- Fusion API What's New.
- AutodeskFusion360/FusionAPIReference.
- Fusion MCP overview.
- Product Help MCP.
- Fusion Data MCP.
- OpenAI Codex plugin and MCP docs.
- MCP protocol specification.

A changed source does not automatically change plugin behavior. Update skills and references only after reading the changed source.
