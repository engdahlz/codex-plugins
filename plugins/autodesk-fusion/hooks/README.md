# Optional Fusion hooks

These are examples for future hook wiring. They are not required for plugin installation and should be reviewed before activation.

Goals:

- Add Fusion runtime/evidence reminders at session start.
- Log MCP/tool intent into the evidence ledger when a run ledger exists.
- Nudge Codex to verify before stopping after significant Fusion changes.

Do not use hooks to bypass platform prompts, steal credentials, or silently mutate unrelated files.
