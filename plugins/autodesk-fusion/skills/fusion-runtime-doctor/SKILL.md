# Fusion runtime doctor

Run or interpret `plugins/autodesk-fusion/scripts/fusion_doctor.py` before live Fusion work when practical.

Use the output to decide whether Codex can work through Product Help MCP, Desktop MCP, Data MCP, Automation API, Computer Use, or repo-only fallbacks.

Report blocked conditions honestly rather than claiming runtime success.
