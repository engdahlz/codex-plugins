# Fusion test and debug

Use for failures, diagnostics, and validation.

## Collect

- Fusion version/channel and Python version.
- active document/product/units state.
- exact command or MCP tool used.
- stack trace and reproduction steps.
- whether live tests, semantic checks, and visual checks passed or were blocked.
