# Fusion Insider compatibility

Use when Axel runs the latest Fusion Insider.

## Rules

- Insider is a test channel, not a public baseline.
- Do not name or describe NDA-only features in public output.
- Store local snapshots under `.fusion-private/` and never commit them.
- Promote support only after public Autodesk docs confirm release status.
