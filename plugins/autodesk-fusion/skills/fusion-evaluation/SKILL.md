# Fusion evaluation

Use for test and release quality.

## Evidence types

- API evidence: source URL, signature, lifecycle.
- runtime evidence: document/product/units/object counts.
- visual evidence: screenshots, appshots, UI state.
- file/cloud evidence: hashes, paths, item/version IDs.

Report unavailable tests as blocked, not passed.
