# Fusion Codex runtime

Use this skill first for broad or ambiguous Fusion requests.

## Route

- Official documentation: Product Help MCP and direct Autodesk/OpenAI URLs.
- Live desktop model work: Fusion Desktop MCP or reviewable Fusion Python add-in.
- Cloud hub/project/item/version data: Fusion Data MCP or MFGDM.
- Headless processing: APS Automation API for Fusion.
- Native UI verification: Codex Computer Use or appshots.
- Palette/Viewer web UI: browser workflows.

## Protocol

Inspect, plan, request approval, mutate only the approved smallest step, then verify. Report unavailable live tests as blocked.
