# Fusion cloud data and MFGDM

Use for cloud data workflows.

## Rules

- Keep cloud metadata separate from active desktop model state.
- Confirm Autodesk identity, hub, project, item, and version scope.
- Do not mutate cloud state without explicit approval.
- Record GraphQL/schema evidence before implementation.
