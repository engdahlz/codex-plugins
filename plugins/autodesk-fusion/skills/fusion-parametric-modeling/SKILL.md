# Fusion parametric modeling

Use for model creation or modification.

## Rules

- Inspect units and use explicit unit expressions.
- Prefer named user parameters for important dimensions.
- Keep sketches constrained and features named.
- Record expected object counts and timeline effects.
- Distinguish construction geometry, reference geometry, and final bodies.
