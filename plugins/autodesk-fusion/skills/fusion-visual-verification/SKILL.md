# Fusion visual verification

Use for UI and presentation verification.

## Rules

- Screenshots prove visual state, not topology or dimensions.
- Combine visual evidence with semantic API evidence for model changes.
- Check camera framing, warnings, dialogs, command inputs, palette DOM, and overlays.
